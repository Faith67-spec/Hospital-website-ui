# 🌐 Full Web Development Internship Projects

## 🧑‍💻 About This Repository
This repository contains projects developed during my Full Web Development Internship at Reccsar Pvt. Ltd. The focus was on building responsive, user-friendly, and visually appealing web interfaces using HTML5 and CSS3.

## 🚀 Projects

### 🏥 Hospital Management Website UI
A modern, responsive hospital website interface designed with a focus on usability and clean layout.

**Features:**
- Semantic HTML5 structure  
- Flexbox & CSS Grid layouts  
- Dark mode UI 🌙  
- CSS animations & hover effects  
- Multi-page navigation  
- Fully responsive (Mobile + Desktop 📱💻)

### 🛒 Responsive E-Commerce Website UI
A clean and interactive e-commerce interface optimized for all screen sizes.

**Features:**
- Product card layout  
- Responsive grid system  
- Hover effects for better UX  
- Mobile-friendly design  

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- Flexbox  
- CSS Grid  
- Google Fonts  

## 📱 Responsive Design
All projects are designed to work seamlessly across:
- Desktop 🖥️  
- Tablet 📲  
- Mobile 📱  

## 📚 What I Learned
- Building structured layouts using HTML5 semantic tags  
- Creating responsive designs using Flexbox & Grid  
- Improving UI/UX with animations and design consistency  
- Writing clean and maintainable front-end code
